General Instructions

Play:

Clicking the play button will open a dialog asking for user input of a username between 1 and 8 characters. After entering this the screen will change to give an option of the 3 game levels.

Instructions:

Shows a picture explaining that clicking will move forward, this will be followed by 3 screens with screenshots showing how to play. This will then return to the main menu.

Scores:

Displays the high scores current;y contained in the high scores file, there is an option to return to the main menu or print the high scores which will print them out on a paper with out logo.

Exit:

This exits the game, but keep it lowkey because nobody should ever stop playing.
